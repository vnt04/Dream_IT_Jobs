# Description:
Build a website to find IT jobs.

Get ideas from ITviec, Topdev and TopCV.

# Tech stack in use:
Backend: Node.js (Express.js), JWT + OAuth2 (for login with social)

Frontend: React, Redux, Tailwindcss

Database: MongoDB
